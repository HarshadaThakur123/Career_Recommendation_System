## **Project Introduction**

The **Career Recommendation System** is a data science-based application that helps students identify suitable career streams based on their academic performance, psychometric analysis, and personal interests.  
It uses Machine Learning and Natural Language Processing techniques to provide intelligent career guidance.

---

## **Technologies Used**

- Python  
- Flask  
- Machine Learning (Random Forest, Logistic Regression, XGBoost)  
- BERT (Sentence Transformer)  
- HTML, CSS  
- SQLite Database  

---

## **Roles and Responsibilities**

- Designed and developed the Career Recommendation System  
- Collected and preprocessed datasets  
- Trained and tested machine learning models  
- Integrated models with Flask web application  
- Created user interface for career prediction  
- Implemented login and history tracking system  
- Documented project and prepared reports
